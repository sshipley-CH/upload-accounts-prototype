Upload package accounts to Companies House Use this service to file your iXBRL package accounts to Companies House. These must be packaged using a zip file format. In this service, you can upload the following package accounts:

  • UKSEF
  • Welsh/English
  • Community Interest Company
  • Overseas
  • Audit Exempt Subsidiary
  • Filing Exempt Subsidiary
  • Limited Partnership
  • Group Package (s400/s401)

Please read the Technical Interface Specification on how to structure your package accounts into a zip file: Hyperlink to TIS. You can find details of all available taxonomies and guidance for filers and developers on the Financial Reporting Council (FRC) website.
